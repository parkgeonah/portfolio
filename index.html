About Me

“끝까지 파고드는 문제 해결이 있는 보안을 선택했습니다.”

외식조리경영 전공에서 진로를 전환해 보안 분야에 입문했습니다. bWAPP/Bee-Box 환경에서 SQL Injection, XSS, 파일 업로드 취약점을 직접 재현하며 공격자 관점의 사고를 훈련해 왔습니다.

단순 PoC에 그치지 않고 DB 조회·조작, 웹쉘 업로드, 세션 하이재킹까지 전체 공격 흐름을 따라가며 공격과 방어를 함께 이해하는 데 집중했습니다.

최종 프로젝트인 SAFE DUGOUT / BullpenTalk에서는 웹 서비스를 직접 개발하고, 동일 환경에 대해 시나리오 기반 모의해킹과 취약점 진단 보고서를 수행했습니다.
