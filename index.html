About Me

“반복적인 업무 대신, 끝까지 파고드는 문제 해결이 있는 보안을 선택했습니다.”

외식조리경영 전공에서 시작해, 보안 사고 뉴스를 계기로 진로를 전환했습니다. bWAPP와 Bee-Box 환경에서 SQL Injection, XSS, 파일 업로드 취약점 등을 직접 재현하며 공격자의 시각에서 웹 서비스를 바라보는 훈련을 해왔습니다.

특히 파일 업로드 취약점, SQL Injection, 세션 고정, 평문전송 등은 단순히 PoC로 끝내지 않고, 실제로 DB 조회·조작, 웹쉘 업로드, 세션 하이재킹까지 전 과정을 따라가며 공격과 방어 모두를 이해하려고 노력했습니다.

최근에는 SAFE DUGOUT / BullpenTalk 프로젝트를 통해 커뮤니티·중고거래·포인트 시스템이 있는 웹 서비스를 직접 개발하고, 동일 환경에 대해 모의해킹·보고서를 작성하며 “개발–공격–방어” 전체 흐름을 경험하고 있습니다.

bWAPP / Bee-Box Pentest Lab

OWASP Top 10 중심 웹 취약점 실습 & 보고서

Bee-Box 환경에서 SQL Injection, XSS, CSRF, 파일 업로드, 세션 관리 취약점 등 다양한 공격 기법을 직접 재현하고, 각 취약점에 대해 개요·공격 시나리오·영향도·개선 권고를 포함한 진단 보고서를 작성했습니다.

  • Burp Suite, OWASP ZAP, Nikto, Nmap 등 도구 활용
  • SQLi (Boolean, Error, UNION, Time-based) 전 유형 재현
  • 웹쉘 업로드 후 Meterpreter 세션 획득까지 전체 흐름 수행
  • 주요정보통신기반시설 가이드와 매핑한 점검 항목 정리
